# Senior Frontend Developer - Web3

## 职位概述

我们正在寻找一位经验丰富的前端开发者加入我们的Web3团队，负责构建下一代去中心化应用的用户界面和用户体验。

## 主要职责

- 设计和开发高性能、响应式的Web3应用前端
- 与智能合约交互，实现钱包连接和区块链操作
- 优化应用性能，确保在各种设备上的流畅体验
- 与设计团队和产品团队紧密合作，实现用户需求
- 参与代码审查，维护代码质量和最佳实践

## 💻 技术要求

### 核心技能
- **前端框架**: React 18+, TypeScript, Next.js
- **状态管理**: Redux Toolkit, Zustand, React Query
- **样式方案**: Tailwind CSS, CSS Modules, Styled Components
- **构建工具**: Vite, Webpack, Rollup

### Web3技能
- **区块链交互**: ethers.js, web3.js, wagmi
- **钱包集成**: MetaMask, WalletConnect, Coinbase Wallet
- **智能合约**: Solidity基础, ABI集成, 事件监听
- **DeFi协议**: Uniswap, Aave, Compound等协议集成经验

### 加分技能
- **性能优化**: React.memo, useMemo, useCallback, 虚拟滚动
- **测试**: Jest, React Testing Library, Cypress
- **CI/CD**: GitHub Actions, Docker, AWS
- **移动端**: React Native, PWA开发经验

## 项目亮点

- **DeFi仪表板**: 构建多链资产管理和收益追踪界面
- **NFT市场**: 开发去中心化NFT交易平台
- **跨链桥**: 实现不同区块链间的资产转移界面
- **治理平台**: 开发DAO投票和提案管理系统

## 我们提供

- **有竞争力的薪资**: 25K-45K RMB/月 + 期权激励
- **灵活工作模式**: 支持远程办公，弹性工作时间
- **技术成长**: 参与前沿Web3技术，学习最新DeFi协议
- **团队文化**: 扁平化管理，鼓励创新和实验
- **福利待遇**: 五险一金，年终奖金，技术培训

## 工作地点
