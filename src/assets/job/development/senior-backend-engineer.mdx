# Senior Backend Engineer - Web3 Infrastructure

## 🚀 职位概述

我们正在寻找一位经验丰富的后端工程师加入我们的Web3基础设施团队，负责构建高可用、高性能的区块链后端服务和API系统。

## 🎯 主要职责

1. 设计和开发高并发、低延迟的Web3后端服务
2. 构建区块链数据索引和API服务，支持多链数据查询
3. 开发智能合约事件监听和数据处理系统
4. 设计和实现微服务架构，确保系统可扩展性
5. 与前端团队合作，提供稳定可靠的API接口
6. 参与系统架构设计，制定技术方案和最佳实践

## 💻 技术要求

### 核心技能
1. **后端语言**: Node.js, Go, Python (至少精通其中两种)
2. **数据库**: PostgreSQL, Redis, MongoDB, InfluxDB
3. **消息队列**: RabbitMQ, Apache Kafka, Redis Streams
4. **容器化**: Docker, Kubernetes, Docker Compose

### Web3技能
1. **区块链交互**: Web3.js, ethers.js, 多链RPC集成
2. **智能合约**: Solidity事件监听, ABI解析, 合约调用
3. **区块链数据**: 区块解析, 交易监控, 事件索引
4. **DeFi协议**: 理解DeFi协议架构, 数据聚合

### 系统架构
1. **微服务**: 服务拆分, 服务发现, 负载均衡
2. **API设计**: RESTful API, GraphQL, gRPC
3. **性能优化**: 缓存策略, 数据库优化, 异步处理
4. **监控运维**: Prometheus, Grafana, ELK Stack

### 加分技能
1. **云服务**: AWS, GCP, Azure, 云原生架构
2. **DevOps**: CI/CD, GitOps, Infrastructure as Code
3. **安全**: API安全, 数据加密, 访问控制
4. **测试**: 单元测试, 集成测试, 性能测试

## 🎨 项目亮点

1. **多链数据索引**: 构建支持Ethereum, Polygon, BSC等链的数据索引服务
2. **实时价格API**: 开发DeFi代币价格聚合和实时更新服务
3. **智能合约监控**: 实现合约事件实时监听和数据处理
4. **用户行为分析**: 构建用户交易行为分析和推荐系统

## 🌟 我们提供

3. **技术成长**: 参与前沿Web3基础设施开发，学习最新技术栈
4. **团队文化**: 扁平化管理，鼓励技术创新和架构优化
